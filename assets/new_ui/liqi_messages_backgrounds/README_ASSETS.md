# LiQi Messages visual backgrounds

Decorative artwork for the Messages and Conversation redesign.

These files are presentation-only. They do not encode conversation, relationship,
session, unread, presence, or membership authority. Runtime product state continues to
come from the existing Messages contracts, repositories, transports, and query layer.

Metro ownership lives in `src/features/messages/screens/messages-redesign-assets.ts`.
The app applies semantic scrims, surfaces, borders, and radii in code.
